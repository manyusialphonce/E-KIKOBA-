# E-KIKOBA

**E-KIKOBA** is a web-based application designed to provide efficient financial management, savings, and record-keeping for users, particularly for cooperative societies, self-help groups, or individual users. The system allows users to track contributions, loans, and transactions, making financial management more transparent and organized.

---

## Features

- **User Management:** Register and manage members or users of the system.  
- **Savings & Contributions:** Track user deposits and contributions.  
- **Loan Management:** Apply, approve, and monitor loans for members.  
- **Transaction History:** Maintain detailed records of all financial transactions.  
- **Reports & Summaries:** Generate reports for savings, loans, and overall financial health.  
- **Admin Dashboard:** Manage users, track transactions, and oversee the system efficiently.  
- **Responsive Design:** Works on mobile, tablet, and desktop devices.  

---

## Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server Environment:** XAMPP (Apache + MySQL)  
- **Optional Libraries:** jQuery, Chart.js (for financial graphs), FontAwesome (for icons)  

---

## Purpose of the Project

The main objectives of **E-KIKOBA** are:  

1. Provide a digital solution for **financial tracking** for cooperatives and self-help groups.  
2. Reduce manual record-keeping errors and make financial data transparent.  
3. Allow members to monitor their contributions, loans, and balances.  
4. Serve as a practical example for students and developers learning **PHP, MySQL, and web applications**.
